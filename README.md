Jeu développé en Java inspiré du Roguelike Backpack Hero, le jeu possède une interface graphique faite avec la bibliothèque Zen.

Différentes fonctionnalités du jeu : 

- Combats et gestion de ceux-ci.
- Inventaire et gestion du sac à dos.
- Gestion des équipements.
- Trésor, marchand, soigneur, etc..
- Hall Of Fame résumant les meilleurs parties faites.
- Gestion des niveaux, des récompenses, etc..

Pour jouer au jeu : 

- Avec le .jar, lancer le jeu depuis le terminal avec : java -jar BackpackHero.jar

Pour trouver comment fonctionne le jeu et comment y jouer, vous pouvez accéder au user.pdf dans le dossier docs/

Pour la javadoc, compiler et nettoyer ces derniers :

- ant javadoc
- ant compile
- ant clean
- ant jar si vous perdez le .jar qui est déjà dans la racine du projet
